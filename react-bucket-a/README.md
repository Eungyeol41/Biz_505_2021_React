# react bucket list project

- SPA(Single Page App)로 구현하는 나의 버킷 리스트
- React를 사용하여 화면 구현
- Nodejs + MongoDB를 연동한 서버구현
- React + Nodejs + MongoDB를 연동하는 API App으로 구현하기

## CRUD

- 입력창에 버킷리스트 입력 후 Enter -> INSERT
- FLAG 클릭 -> 긴급, 매우 중요, 중요, 일반이 번갈아 표시됨
- 완료 항목 클릭 -> 완료일자 등록
- 취소 체크 -> 취소선
- 항목 클릭 -> UPDATE
